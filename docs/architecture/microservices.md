# Microservices Architecture

ZarishSphere Platform consists of 14 independent microservices.

## Service List

1. **zarish-fin** - Financial Management
2. **zarish-hra** - Human Resources
3. **zarish-his** - Hospital Information System
4. **zarish-connect** - API Gateway
5. **zarish-sphere** - FHIR Core
6. **zarish-access** - Authentication
7. **zarish-terms** - Terminology
8. **zarish-forms** - Form Builder
9. **zarish-geo** - Geography
10. **zarish-support** - Support Ticketing
11. **zarish-management** - Project Management
12. **zarish-wiki** - Knowledge Base
13. **zarish-insight** - Analytics
14. **zarish-log** - Logistics

See [Architecture Overview](overview.md) for detailed diagrams.
